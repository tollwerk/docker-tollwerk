CSS Classes

frame
The frame class restricts the width of an element to the global maximum width, gives it an automatic horizontal padding and centers it into the viewport.
informal
Use the informal CSS class to remove the user agent styling from an element.
  • Removes bullet points, numbering, margin and padding from ordered (<ol>) and unordered (<ul>) lists.
hide-element
Use this class to visually hide an element. The element will stay visible for assistive technology.
hide-content
Use this class in combination with a fixed width to visually hide the content of an element. The element itself will stay visible (also for assistive technology), but its contents will be shifted beyond the element boundaries.
